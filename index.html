potionshop

click the grid overlay to start audio

a browser-based FM burst sequencer — six channels each firing rhythmic bursts of repeated FM tones. every parameter follows its own looping sequence, so patterns drift and phase independently.

parameters · row 7, cols 0–5

div
burst divisor 1–32 · smaller = faster
reps
repetitions per burst · 1–8 or ∞
note
scale degree · default scale is major
level
amplitude · 0–1
harm
FM ratio 2–4 · integers = harmonic · non-integers = metallic
env
envelope shape · 0 = snappy · 1 = slower decay

tap a param button again to toggle A ↔ B layer (B is an additive offset over A · slow-strobes when active)

step editing · rows 0–5

left cols 0–7 = A layer · right cols 8–15 = B layer
tap a step → value picker appears on rows 0–1
tap past the last step → extend the sequence
tap a step again inside the picker → delete it

actions · row 7, cols 9–15

CLR · 9
arm → tap channel → clear selected param's A+B
LOCK · 10
arm → tap channels to toggle locked mode — keeps all param lengths equal · on by default
RND · 14
arm → tap channel → randomize all A-layer params
MUT · 15
arm → tap channel → nudge A-layer values ±25%

launch & modes · row 6

0–5
tap to launch or stop each channel · channels snap to the launch grid (default: next quarter note)
KB · 12
keyboard mode · tap values directly to build sequences gesture by gesture
PROB · 13
burst probability per channel · cols 0–14 = 0–100% · col 15 = per-burst vs per-hit
QNT · 14
opens scale + quantize picker · row 0: scale · row 1: 1–16 events per whole note
SND · 15
envelope timing (shape / burst / hit) + geode amplitude mode (off / transient / sustain / cycle)

keyboard mode

page 1 · rows 0–1 note · rows 2–3 div · rows 4–5 reps
page 2 · rows 0–1 level · rows 2–3 harm · rows 4–5 env
row 6: scale selector (8 options) · row 7: channel select
col 12 commit+exit · col 13 page toggle · col 14 clear buffers

scales + quantize · row 7 col 6 (or QNT)

row 0: chromatic · major · minor · pentatonic · dorian · akebono · hijaz · kurd · bayati · rast · zen · wuSheng
row 1: quantize 1–16 events per whole note

locked mode

all channels start locked — adding or removing a step on any parameter extends or trims all others to match. arm LOCK (row 7 col 10) and tap a channel to toggle. randomize also respects locked state.

commands

launch(ch, div?, reps?, note?, level?, harm?, env?) — start or replace channel ch (1..6)
stop(ch) · stopAll() — silence one channel or everything
s([1,2,3]) — make a sequins (cycles forever)
scales.major / minor / pentatonic / dorian / chromatic
scales.akebono / hijaz / kurd / bayati / rast / zen / wuSheng — eastern + percussion
refresh() — redraw the grid after direct state writes

direct state

engine.channels[0].note = s([0,4,7]) — swap a single param live
setScale(scales.akebono) or engine.scale = scales.akebono; refresh() — set global scale (shared by all channels)
engine.quantize = 16 — per-fire snap grid (0 = off · events per whole note)
engine.launchGrid = 4 — launch alignment grid (the global clock pulse · 0 = off)

examples

launch(1, 8, 4, 2)
setScale(scales.major); launch(1, s([4,3]), -1, s([0,4,7]), 0.7)
launch(1, 8, s([4,3]), s([0,4,7]), s([1,0.5]), s([2,2,7])) — harm sequins at position 6

browsers block audio until a user gesture · this enables web audio for the session
